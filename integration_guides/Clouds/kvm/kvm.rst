KVM
---

Overview
^^^^^^^^^
Kernel-based Virtual Machine (KVM) is an open source virtualization technology built into LinuxÂ®. Specifically, KVM lets you turn Linux into a hypervisor that allows a host machine to run multiple, isolated virtual environments called guests or virtual machines (VMs).

Features
^^^^^^^^
* CEPH storage support
* KVM migration workflow
* OnApp migration support
* Backup/restore for local datastores and LVM
* Caching virtual machines includes metadata on network and storage volumes attached to a KVM VM
* VM Actions, MultiDisk, Reconfigure, Apps & Blueprints.
* Backups/Restore added
* VM Inventory
