Centralized
------------

In the Single Node/All-in-one configuration, all components required for Morpheus are installed and configured during the Morpheus ```reconfigure`` command, including:

Appliance Host
 - Application Tier
    - Morpheus App
 - Database Tier
    - MySQL
 - Message Tier
    - RabbitMQ
 - Search Tier
    - Elasticsearch
